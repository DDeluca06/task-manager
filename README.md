### Features

### Tech Stack
- **Frontend**: Bootstrap
- **Backend**: Node.js/Sequelize/Express.JS
- **Database**: MySQL

### Installation
1. Clone the repository
2. Run `npm install` to install dependencies
3. Run `npm start` to start the server
4. Open your browser and navigate to `http://localhost:3000`
